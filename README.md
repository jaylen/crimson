# Crimson

A lightweight dependency injection for Java.

## Usage

Use as Maven dependency.

```xml
<dependency>
    <groupId>org.jaylen.crimson</groupId>
    <artifactId>crimson</artifactId>
    <version>0.1.0-SNAPSHOT</version>
</dependency>
```

Requires JDK 8.0 or above.
